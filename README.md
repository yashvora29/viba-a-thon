# 🍽️ Restaurant Inventory AI Agent

An **AI-powered restaurant inventory manager** that monitors ingredient stock, checks recipe feasibility, detects low-stock items, and automatically generates supplier orders.

Built for **Vibe-A-Thon Hackathon**.

## 🚀 Features
- Check current inventory
- Detect low-stock ingredients
- Analyze if a dish can be cooked
- Generate supplier restock orders
- Simulated micropayment authorization (x402)

## ⚙️ Run the Project
```bash
npm install
node agent.js
